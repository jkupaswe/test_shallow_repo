# test_shallow_repo

Small Python example showing a reusable `main()` function that prints different kinds of input to the console, plus an `interface.py` runner with example parameter sets.

## Run

- CLI usage:
	- `python main.py --text "hello"`
	- `python main.py --number 123`
	- `python main.py --json '{"a": 1, "b": [2, 3]}'`
	- `python main.py --items a,b,c`
	- stdin echo: `echo "hi" | python main.py`

- Run the parameterized examples:
	- `python interface.py`
